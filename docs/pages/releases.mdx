# Foundry v1.3.1

Foundry v1.3.1 is a bugfix release for anvil, cast and forge.

## Anvil Fixes
- fix(anvil): unwrap panic in eth/backend/mem/mod.rs ([#11141](https://github.com/foundry-rs/foundry/pull/11141)) by [@zerosnacks](https://github.com/zerosnacks)  
- fix: use empty 7702 auths ([#11182](https://github.com/foundry-rs/foundry/pull/11182)) by [@mattsse](https://github.com/mattsse) 

## Cast Fixes
- fix(cast): use all providers if local artifacts used ([#11090](https://github.com/foundry-rs/foundry/pull/11090)) by [@grandizzy](https://github.com/grandizzy)  
- fix(cast): improve handling of mktx --raw-unsigned with runtime validation ([#11111](https://github.com/foundry-rs/foundry/pull/11111)) by [@mablr](https://github.com/mablr) 

## Forge Fixes
- fix(forge): consistent handling unresolved imports ([#11164](https://github.com/foundry-rs/foundry/pull/11164)) by [@grandizzy](https://github.com/grandizzy)  
- fix(lint): exclude disabled ids in multi-lint passes ([#11122](https://github.com/foundry-rs/foundry/pull/11122)) by [@0xrusowsky](https://github.com/0xrusowsky)  
- fix(forge): bump compilers to fix remapping sol file ([#11184](https://github.com/foundry-rs/foundry/pull/11184)) by [@grandizzy](https://github.com/grandizzy) 
- fix(coverage): disambiguate duplicate function names ([#11188](https://github.com/foundry-rs/foundry/pull/11188)) by [@DaniPopes](https://github.com/DaniPopes) 
- fix(forge): set state changes journal in active fork ([#11196](https://github.com/foundry-rs/foundry/pull/11196)) by [@grandizzy](https://github.com/grandizzy)
- fix(forge): improve error messages for etherscan verification failures ([#11194](https://github.com/foundry-rs/foundry/pull/11194)) by [@shiyasmohd](https://github.com/shiyasmohd) 
- fix(fmt): don't panic on stdin read failure ([#11226](https://github.com/foundry-rs/foundry/pull/11226)) by [@DaniPopes](https://github.com/DaniPopes) 
- fix(forge script): repeated vm.createSelectFork with same RPC causes segfault ([#11250](https://github.com/foundry-rs/foundry/pull/11250)) by [@zerosnacks](https://github.com/zerosnacks)
