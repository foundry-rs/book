## `parseAddress`

### Signature

```solidity
function parseAddress(string calldata stringifiedValue) external pure returns (address parsedValue);
```

### Description

Parses a hex string into an `address`.

### Parameters

| Parameter          | Type     | Description              |
|--------------------|----------|--------------------------|
| `stringifiedValue` | `string` | The hex string to parse  |

### Returns

| Type      | Description         |
|-----------|---------------------|
| `address` | The parsed address  |

### Examples

```solidity [test/ParseAddress.t.sol]
string memory addrString = "0x7109709ECfa91a80626fF3989D68f67F5b1DD12D";
address result = vm.parseAddress(addrString);
// result = 0x7109709ECfa91a80626fF3989D68f67F5b1DD12D
```
