## cast --to-ascii

### NAME

cast---to-ascii - Convert hex data to an ASCII string.

### SYNOPSIS

``cast --to-ascii`` [*options*] *text*

### DESCRIPTION

Convert hex data to an ASCII string.

### OPTIONS

{{#include common-options.md}}

### EXAMPLES

1. Convert hex data to an ASCII string:
    ```sh
    cast --to-ascii "0x68656c6c6f"
    ```
