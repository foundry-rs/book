## cast 4byte

### NAME

cast-4byte - Get the function signatures for the given selector from <https://sig.eth.samczsun.com>.

### SYNOPSIS

``cast 4byte`` [*options*] *sig*

### DESCRIPTION

Get the function signatures for the given selector from <https://sig.eth.samczsun.com>.
