## cast help

### NAME

cast-help - Get help for a Cast command

### SYNOPSIS

``cast help`` [*subcommand*]

### DESCRIPTION

Prints a help message for the given command.

### EXAMPLES

1. Get help for a command:
    ```sh
    cast help call
    ```

2. Help is also available with the `--help` flag:
    ```sh
    cast call --help
    ```
