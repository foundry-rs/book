## probe help

### NAME

probe-help - Get help for a Probe command

### SYNOPSIS

``probe help`` [*subcommand*]

### DESCRIPTION

Prints a help message for the given command.

### EXAMPLES

1. Get help for a command:
    ```sh
    probe help call
    ```

2. Help is also available with the `--help` flag:
    ```sh
    probe call --help
    ```
