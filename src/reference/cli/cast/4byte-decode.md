# cast 4byte-decode

Decode ABI-encoded calldata using https://openchain.xyz

```bash
$ cast 4byte-decode --help
```

```txt
Usage: cast 4byte-decode [OPTIONS] [CALLDATA]

Arguments:
  [CALLDATA]  The ABI-encoded calldata

Options:
  -h, --help  Print help

Display options:
  -j, --json  Print the decoded calldata as JSON
```
