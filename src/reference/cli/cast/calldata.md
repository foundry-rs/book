# cast calldata

ABI-encode a function with arguments

```bash
$ cast calldata --help
```

```txt
Usage: cast calldata <SIG> [ARGS]...

Arguments:
  <SIG>      The function signature in the format
             `<name>(<in-types>)(<out-types>)`
  [ARGS]...  The arguments to encode

Options:
  -h, --help  Print help
```
