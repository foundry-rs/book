# probe from-wei

Convert wei into an ETH amount.

```bash
$ probe from-wei --help
Usage: probe from-wei [VALUE] [UNIT]

Arguments:
  [VALUE]
          The value to convert

  [UNIT]
          The unit to convert from (ether, gwei, wei)
          
          [default: eth]

Options:
  -h, --help
          Print help (see a summary with '-h')
```
