# probe to-unit

Convert an ETH amount into another unit (ether, gwei or wei).

```bash
$ probe to-unit --help
Usage: probe to-unit [VALUE] [UNIT]

Arguments:
  [VALUE]
          The value to convert

  [UNIT]
          The unit to convert to (ether, gwei, wei)
          
          [default: wei]

Options:
  -h, --help
          Print help (see a summary with '-h')
```
