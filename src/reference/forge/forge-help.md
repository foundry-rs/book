## forge help

### NAME

forge-help - Get help for a Forge command

### SYNOPSIS

``forge help`` [*subcommand*]

### DESCRIPTION

Prints a help message for the given command.

### EXAMPLES

1. Get help for a command:
    ```sh
    forge help build
    ```

2. Help is also available with the `--help` flag:
    ```sh
    forge build --help
    ```
