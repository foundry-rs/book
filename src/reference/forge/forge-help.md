## spark help

### NAME

spark-help - Get help for a Spark command

### SYNOPSIS

``spark help`` [*subcommand*]

### DESCRIPTION

Prints a help message for the given command.

### EXAMPLES

1. Get help for a command:
    ```sh
    spark help build
    ```

2. Help is also available with the `--help` flag:
    ```sh
    spark build --help
    ```
