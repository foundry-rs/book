## cast estimate

### NAME

cast-estimate - Estimate the gas cost of a transaction.

### SYNOPSIS

`cast estimate` [*options*] _to_ _sig_ [*args...*]

### DESCRIPTION

Estimate the gas cost of a transaction.

The destination (_to_) can be an ENS name or an address.
