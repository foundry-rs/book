## cast wallet verify

### NAME

cast-wallet-verify - Verify the signature of a message.

### SYNOPSIS

`cast wallet verify` [*options*] `--address` _address_ _message_ _signature_

### DESCRIPTION

Verify the signature of a message.

### OPTIONS

#### Signature Options

`-a` _address_  
`--address` _address_  
&nbsp;&nbsp;&nbsp;&nbsp;The address of the message signer.
