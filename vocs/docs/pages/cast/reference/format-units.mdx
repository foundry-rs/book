## cast format-units

Format a number from smallest unit to decimal with arbitrary decimals.

Examples: - 1000000 6       (for USDC, result: 1.0) - 2500000000000 12 (for 12
decimals, result: 2.5) - 1230 3          (for 3 decimals, result: 1.23)

```bash
$ cast format-units --help
```
