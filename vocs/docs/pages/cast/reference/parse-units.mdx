## cast parse-units

Convert a number from decimal to smallest unit with arbitrary decimals.

Examples: - 1.0 6    (for USDC, result: 1000000) - 2.5 12   (for 12 decimals
token, result: 2500000000000) - 1.23 3   (for 3 decimals token, result: 1230)

```bash
$ cast parse-units --help
```
