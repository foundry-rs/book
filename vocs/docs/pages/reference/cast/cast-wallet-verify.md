## cast wallet verify

### NAME

cast-wallet-verify - Verify the signature of a message.

### SYNOPSIS

``cast wallet verify`` [*options*] `--address` *address* *message* *signature*

### DESCRIPTION

Verify the signature of a message.

### OPTIONS

#### Signature Options

`-a` *address*  
`--address` *address*
