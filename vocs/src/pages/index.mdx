## Get Started

::::steps

### Install Foundry

```bash
curl -L https://foundry.paradigm.xyz | bash
foundryup
```

### Create a project

```bash
forge init hello_foundry
cd hello_foundry
```

### Build and test

```bash
forge build
forge test
```
